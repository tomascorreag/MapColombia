# Derived datasets — provenance and licenses

Every file in this directory is a reproducible transformation of third-party
open data, generated by `pipeline/build_frontend_data.py` (no manual edits).
Transformations applied: filtering with documented exclusion counts (never
imputation), recoding to compact columnar/JSON layouts, geometry
simplification, and joins on DANE/DIVIPOLA codes. Per-file method details are
embedded in each file's `meta` block.

The repository's MIT license does **not** cover these files; the terms below
do. None of the source institutions endorses this project.

## Per-file terms

### `violence.bin` + `violence_meta.json` + `violence_details.bin`
- **Source**: CNMH — SIEVCAC, Observatorio de Memoria y Conflicto, corte
  2026-03-31, via the CNMH ArcGIS Geoportal
  (https://geoportal-de-datos-abiertos-cnmh-cnmh.hub.arcgis.com/). The
  `casos_*` files feed `violence.bin`; the `victimas_*` files feed the
  per-event victim portrait in `violence_details.bin`.
- **Source license**: [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/),
  stated per item; verified 2026-06-10
  (`docs/evidence/cnmh-agol-license-2026-06-10.json`).
- **Attribution**: Centro Nacional de Memoria Histórica (CNMH), Observatorio
  de Memoria y Conflicto. *Material modificado* (filtered, recoded to binary
  columnar; exclusions documented in `violence_meta.json`).
- **`violence_details.bin`** is gi-aligned to `violence.bin` (same row order)
  and holds *aggregate* victim tallies per case — civilian/combatant totals,
  sex, age bucket, fatal/non-fatal, and the most-frequent recorded occupation
  (layout + occupation table in `violence_meta.json` → `details`). Counts are of
  individually-recorded victims; blank attributes are uncounted, never imputed.
  **No victim names** — SIEVCAC does not publish them and neither does this file.
- **These derived files**: [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).

## Integrity note

This project records real victims and real elections. Missing values are
never estimated or imputed; excluded records are counted and disclosed in
each file's `meta`. Violence data carries the CNMH case ID per event as its
citation; no victim names are included in any file here.
